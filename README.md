# Video Content Suite

视频内容套件 - 用于视频内容创作、管理和发布的工具集合。

## 项目简介

本项目是一个视频内容创作和管理的工作空间，包含各种工具、配置和文档，帮助高效地进行视频内容的生产和运营。

## 目录结构

- `AGENTS.md` - 工作空间配置指南
- `SOUL.md` - 运营专家角色定义
- `IDENTITY.md` - 身份配置
- `memory/` - 记忆和日志存储
- `video-content-suite/` - 视频内容相关文件
- `proactivity/` - 主动任务配置
- `self-improving/` - 自我改进相关配置

## 使用说明

1. 查看 `AGENTS.md` 了解工作空间配置
2. 查看 `task_plan.md` 了解当前任务计划
3. 查看 `progress.md` 了解项目进展
